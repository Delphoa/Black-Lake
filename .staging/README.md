# .staging

Temporary holding area primarily for incomplete, unverified, or in-progress knowledge artifacts.
Staging also includes temporary records, or reference pointers for regular and planned tasks.

Do not treat staged material as canonical. Move complete work into `.lake-data/`, `.reports/`, or `.logs/` after validation.

## Durable Pointer Records

`.staging/arxiv-dep-dedup-index.json` is an exception to the temporary-only rule. It is a public-safe pointer index for Arxiv DEP deduplication and should be updated after each Arxiv DEP run with repository-relative artifact paths only.
