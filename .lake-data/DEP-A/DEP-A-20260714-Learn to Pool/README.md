## Insights and Relevance

Pooling-aware fine-tuning prevents catastrophic aggregation loss and multi-factor training supports 50–83% vector reduction with one encoder. The review flags mostly single runs, possible evaluation-based checkpoint selection, 7–13x clustering-training overhead, and missing complete index economics.
