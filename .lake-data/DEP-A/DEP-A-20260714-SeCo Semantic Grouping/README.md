## Insights and Relevance

SeCo’s durable idea is query-conditioned, noncontiguous grouping. The review finds that a positive query-relevance multiplier cannot change the assignment argmax, a negative one reverses it, and the full system still uses position-conditioned states. Large reported gains therefore require stronger split, seed, uncertainty, and artifact evidence.
